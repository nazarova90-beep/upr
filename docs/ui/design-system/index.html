Semantic tokens

Palette colours are the raw paint. Semantic tokens assign each colour a job in the UI — background, text, accent, danger. Components only reference these roles, so re-skinning the whole product later is a matter of repointing tokens, not editing every component.

Spacing

The spacing scale is built on a 4 px base unit. Every value is a multiple of 4, which keeps paddings, gaps and margins visually consistent across components. Use the token names (e.g. --space-md) instead of hard-coded pixel values.

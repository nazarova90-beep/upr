# Product Specs — оглавление

**Сначала** прочитайте **`../product.md`** — итоговая **продуктовая стратегия** без сценариев и UI (столпы, границы MVP, монетизация, риски, карта документов).

Подход к принятию решений — в `../PRODUCT_SENSE.md`.

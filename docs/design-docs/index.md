## ADR template (Architecture Decision Record)

Each design-doc contains:

1. **Context** — problem, available options.
2. **Options considered** — pros / cons.
3. **Decision**.
4. **Rationale**.
5. **Consequences** — gains and trade-offs.
6. **Related decisions**.
